
# Pokémon Puzzle Battle

A retro-style Match-3 Pokémon battle game built with HTML, CSS, and JavaScript.

## Features

- Match-3 puzzle mechanics
- Pokémon battle animations
- Combo attack system
- Power-ups:
  - Hint
  - Shuffle
  - Bomb
  - Mega Attack
- Local leaderboard
- Retro pixel-art design
- Responsive layout

## Controls

| Key | Action |
|------|---------|
| Q | Hint |
| W | Shuffle |
| E | Mega Attack |
| R | Bomb |
| P | Pause |
| ESC | Pause |

## Screens

- Trainer Login
